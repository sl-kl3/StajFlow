**Modul sahibi:** Zuhal  
**Dosyalar:** `templates/admin/*`

---

## Ne yaptik?

Admin bolumu sistemi yoneten kisinin ekrani. Ogrenci basvuru yapar, admin altyapiyi kurar.

## Admin sayfalari

| Sayfa | Dosya | Islev |
|-------|-------|-------|
| Dashboard | admin/anasayfa.html | Ozet istatistikler |
| Ogrenciler | admin/ogrenciler.html | Kullanici ekle / sil |
| Basvurular | admin/basvurular.html | Tum staj basvurulari |
| Sirketler | admin/sirketler.html | Sirket + staj programi ekle |
| Raporlar | admin/raporlar.html | Sayisal rapor |

## Admin neler yapabilir?

1. Yeni kullanici ekler (ogrenci / danisman / admin)
2. Kullanici siler (kendi hesabi haric)
3. Sirket ekler
4. Staj programi (ilan) acar, kontenjan belirler
5. Ilani aktif/pasif yapar
6. Rapor sayfasinda sayilari gorur
